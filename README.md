# 🌲 Wandering in the Woods
CS251 Project — Lewis University — Spring 2026

A K-8 educational simulation demonstrating random walks and probability.
Two characters are lost in a forest and move randomly — how long until they find each other?

## ▶️ How to Run (No Python needed)
1. Download main.exe from the dist folder
2. Double click it — game opens instantly!
3. If Windows warns you → click More info → Run anyway

## 💻 Run from Source Code
pip install pygame numpy
python main.py

## 📁 Files
- main.py — Main menu
- mode_k2.py — Grades K-2 mode
- mode_35.py — Grades 3-5 mode
- sounds.py — Sound effects
- Game_music.mp3 — Background music
- dist/main.exe — Standalone executable
